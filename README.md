# Ko-Swipe-ML
#### Ko Swipe Recommender System
## Getting Started

`python3.11`
```
git clone https://github.com/{username}/Ko-Swipe-ML.git

cd Ko-Swipe-ML

python -m venv .venv
source .venv/bin/activate # linux or mac
.venv\Scripts\activate # window

pip install -r requirements.txt
```
